CSS-правила

Весь CSS-код складається з блоків, що повторюються: 

        селектор {
          властивість1: значення;
          властивість2: значення;
        }

Такий блок називається «CSS-правило». Кожне CSS-правило містить хоча б один селектор та властивість.
Найпростіші селектори — це селектори по іменам тегів. З їх допомогою можна задати стилі для всіх абзаців на сторінці, для всіх посилань, заголовків першого рівня і так далі. Такі селектори містять ім'я тегу без символів < і >. Наприклад: 

        p {
          /* стилі для абзаців */
        }
        h1 {
          /* стилі для заголовків */
        }
